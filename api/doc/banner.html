banner 栏目首页内容

取得导航信息

请求url：http://112.74.89.94/shspring/index.php/api/banner

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 navigation  每个接口的标示
key String 是 kuanshi  导航key值
page int 否 10 页数

请求示例（款式）

{"tag":"banner","data":{"key":"kuanshi"}}

请求示例（发现）

{"tag":"banner","data":{"key":"faxian"}}

请求示例（二手）

{"tag":"banner","data":{"key":"ershou","page":1}}

请求示例（周边）

{"tag":"banner","data":{"key":"zhoubian","page":1}}
