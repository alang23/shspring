consignment 寄售申请

用户注册

请求url：http://112.74.89.94/shspring/index.php/api/consignment/submit

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 register  每个接口的标示
uid int 是   用户id
token String 是   用户登陆token
name String 是   产品名称
price String 是   产品价格
tel String 是   联系方式
intro String 是   产品描述
pic Array 是   用户上传图片集
--- filedata 是   图片base64值(需经过urlencode处理)
--- extension 是   图片格式(jpg.png..等)

请求示例

{
	"tag":"consignment_submit",
	"data":
	{
		"uid":1,
		"token":"123123123",
		"name":"asd",                 //寄售产品名称
		"price":"asd",                 //寄售产品价格
		"tel":"asd",                 //寄售联系方式
		"intro":"asd",                 //寄售产品说明
		"pic":[                        //图片集
			{
			"filedata":"asdSDFds",
			"extension":"jpg"
			},
			{
			"filedata":"234dfsdf",
			"extension":"png"
			}
		]
	}
}

返回结果示例

{
	"errcode":"0",
	"msg":"200",
	"data":
	{
		"upload_img_num":2    //成功上传图片的数量
	}
}

consignment_list 寄售列表

个人寄售列表

请求url：http://112.74.89.94/shspring/index.php/api/consignment/lists

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 consignment_list  每个接口的标示
uid int 是   用户id
token String 是   用户登陆token

请求示例

{
	"tag":"consignment_list",
	"data":
	{
		"uid":1,                    //用户id           
		"token":"123123"           //用户token
	}
}

返回结果示例

{
"errcode":"0",
"msg":"200",
"data":
{
	"items":
	[
		{
			"id":"1",                                                   //寄售商品id
			"name":"\u5305\u5305",                                      //商品名称
			"img":"http:\/\/www.spring.sp\/uploads\/consignment\/",     //图片地址
			"price":"30.12",                                             //寄售价格
			"consignment_date":"2015-05-06",                             //寄售时间
			"collection":100,                                            //收藏数
			"exposed":200,                                                //浏览数
			"status":"0"                                                  //状态(0-审核中，1-寄售中，2-已出售，3-审核失败)
		}
	]
}
}

consignment_del 删除寄售

删除个人寄售产品

请求url：http://112.74.89.94/shspring/index.php/api/consignment/del

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 consignment_del  每个接口的标示
uid int 是   用户id
token String 是   用户登陆token
id int 是   产品id

请求示例

{
	"tag":"consignment_del",
	"data":
	{
		"uid":1,                    //用户id           
		"token":"123123"           //用户token
		"id":1                     //寄售商品id
	}
}

返回结果示例

{
"errcode":"0",
"msg":"\u64cd\u4f5c\u6210\u529f",
"data":[]
}
