cart 购物车

点击购买

请求url：http://112.74.89.94/shspring/index.php/api/cart/buy

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 buy  每个接口的标示
pid int 是   产品id
uid int 否   用户id
token int 否   当前登陆token
num int 是   数量
enabled int 是   是否勾选(0-未勾选，1--勾选)

请求示例

{
"tag":"buy",
"data":
	{
		"pid":1,
		"uid":1,
		"pid":1,
		"token":"13123",
		"num":1,
		"enabled":1
	}
}

返回结果示例

{
	"errcode":0,
	"msg":"产品添加成功 ",
	"data":
	[]
}
