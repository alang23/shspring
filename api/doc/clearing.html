clearing/submit 提交订单

提交订单

请求url：http://112.74.89.94/shspring/index.php/api/clearing/submit

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 clearing  每个接口的标示
uid int 是   用户id
token String 是   token
exp int 是   配送方式
invoice int 是   是否索取发票
address String 是   收货地址
items Array 是   商品信息
  id int 是   商品id
num int 是   商品数量

请求示例

{
"tag":"clearing_submit",
"data":
	{
	"uid":1,
	"token":"ddddd",
	"exp":1,      //配送类型，1-快递，2-自提，3-送货
	"exp_id":1,      //类型id，没有则为"0"，
	"invoice":1,     //0-不开发票，1-个人，2-单位
	"invoice_title":"发票抬头", //个人发票或者不取发票时为空
	"address":
	{
		"province_id":1, //省份id
		"city_id":1,     //城市id
		"area_id":3,     //
		"detail":"shenzhenfutian"
	},
	"hongbao":
	{
		"id":1,
		"code":"abcde"
	},
	"items":
		[
			{
				"id":1,
				"num":2
			},
			{
				"id":2,
				"num":2
			}
		]
	}
}

返回结果示例

{
"errcode":0,
"msg":"ok",
"data":
{
	"total":2,                  //购买的商品数
	"order_no":"HM1432696913"  //订单号
	"order_id":"12"            //订单id
	"status" : "1"             //状态（1.支付成功，2-支付失败，0-未知错误）
}
}
