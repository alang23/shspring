add_address 添加/修改收货地址

添加收货地址

请求url：http://112.74.89.94/shspring/index.php/api/clearing/add_address

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 register  每个接口的标示
uid int 是   用户id
id int 修改时必须   地址id
token String 是   token
province_id int 是   省份id
city_id int 是   城市id
area_id int 是   用户密码
address String 是   详细地址
mobile String 是   电话
zip_code String 否   邮编
realname String 是   收货人

请求示例

{
	"tag":"add_address",
	"data":
		{
			"uid":1,
			"id":1, //id不为空则为修改
			"token":"123123",
			"province_id":1,
			"city_id":2,
			"area_id":3,
			"address":"\u6df1\u5733\u798f\u7530",
			"mobile":"13800138000",
			"zip_code":"\u90ae\u7f16",
			"realname":"admin"
		}
}

返回结果示例

{
	"errcode":0,
	"msg":"ok",
	"data":
	{
		"id":4 //返回添加的数据id
	}
}
