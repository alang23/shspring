realtime 搜索

用户输入实时返回搜索结果

请求url：http://112.74.89.94/shspring/index.php/api/search/realtime

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 search  每个接口的标示
uid int 否   用户id
kw String 是   用户输入关键字

{
	"tag":"search",

	"errcode":0,
	"msg":"ok",
	"data":
