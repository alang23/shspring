xiu 买家秀

用户提交买家秀

请求url：http://112.74.89.94/shspring/index.php/api/xiu/index

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 xiu  每个接口的标示
uid int 是   用户id
pid int 是   商品id（没有商品关联时为0）
token String 是   用户登陆token
intro String 是   描述
pic Array 是   用户上传图片集
--- filedata 是   图片base64值(需经过urlencode处理)
--- extension 是   图片格式(jpg.png..等)

请求示例

{
	"tag":"xiu",
	"data":
	{
		"uid":1,
		"token":"123123123",
		"intro":"asd",                 //买家秀描述
		"pic":[                        //图片集
			{
			"filedata":"asdSDFds",
			"extension":"jpg"
			},
			{
			"filedata":"234dfsdf",
			"extension":"png"
			}
		]
	}
}

返回结果示例

{
	"errcode":"0",
	"msg":"200",
	"data":
	{
		"upload_img_num":2    //成功上传图片的数量
	}
}
