product_detail 导航

产品详情

请求url：http://112.74.89.94/shspring/index.php/api/product/detail

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 kuanshi  每个接口的标示
id int 是   产品id
uid int 是   用户id(未登陆传0)
token string 是   当前登陆token(未登陆传空)

请求示例

{
"tag":"product_detail",
"data":
	{
	"id":4,
	"uid":1,
	"token":"aaaaaa"
	}
}
