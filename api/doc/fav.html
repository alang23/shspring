fav 收藏

用户收藏产品

请求url：http://112.74.89.94/shspring/index.php/api/fav/do_fav

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 do_fav 每个接口的标示
pid int 是   当前页产品id
uid int 是   用户id
token String 是   用户当前登陆token
act String 是   操作标示(add 添加收藏，del 取消收藏)

请求示例

{
	"tag":"do_fav",
	"data":
		{
			"id":1,
			"uid":1,
			"pid":1,
			"token":"13123",
			"act":"del", //add-添加收藏，del-删除收藏
			
		}
}

返回结果示例

			
{
	"errcode":0,
	"msg":"添加成功 ",
	"data":[]
}

错误提示

	-1   缺少必要参数
	-2   产品不存在
	-3   重复添加
	-4   系统错误
	-5   未登陆
