register 注册

用户注册

请求url：http://112.74.89.94/shspring/index.php/api/register

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 register  每个接口的标示
mobilePhoneNumber String 是   接收短信的手机号
code String 是   手机收到的验证码
passwd String 是   用户密码

请求示例

{
	"tag":"register",
	"data":
	{
		"mobilePhoneNumber":"15814073945",
		"code":"123456",
		"passwd":"123456"
	}
}

返回结果示例

{
	"errcode":0,
	"msg":"注册成功"
}

changepwd 修改密码

修改密码接口

请求url：http://112.74.89.94/shspring/index.php/api/register/changepwd

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 changepwd  每个接口的标示
token String 是   当前token
mobilePhoneNumber String 是   手机号
passwd String 是   原密码
newpasswd String 是   新密码

请求示例

{
	"tag":"changepwd",
	"data":
		{
			"token":"12313123",
			"mobilePhoneNumber":"13800138000",
			"passwd":"123123",
			"newpasswd":"123123"
		}
}

返回结果示例

{
	"errcode":0,
	"msg":"\u77ed\u4fe1\u53d1\u9001\u6210\u529f"
}
