feedback 意见反馈

用户注册

请求url：http://112.74.89.94/shspring/index.php/api/feedback/index

应用级输入参数

名称 类型 是否必须 示例值 默认值 描述
tag String 是 feedback  每个接口的标示
uid int 是   用户id
token String 是   当前登陆token
content String 是   意见内容

请求示例

{
	"tag":"feedback",
	"data":
	{
		"uid":"15814073945",
		"token":"123456",
		"content":"asdfasdfasdf"
	}
}

	"msg":"\u767b\u5f55\u6210\u529f",
